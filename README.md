# ✊✋✌️ Stone Paper Scissors Game

A simple and interactive Stone Paper Scissors game built using HTML, CSS, and JavaScript. Players can compete against the computer, and the game instantly determines the winner based on the selected moves.

## 🚀 Features

* 🎮 Play against the computer
* ⚡ Instant result generation
* 📊 Score tracking
* 🎨 Clean and responsive user interface
* 🔄 Play multiple rounds
* 📱 Mobile-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure


stone-paper-scissors/
├── index.html
├── style.css
├── script.js
└── README.md


## 🎯 How to Play

1. Choose Stone, Paper, or Scissors.
2. The computer will randomly select its move.
3. The winner is determined by the game rules:

   * Stone beats Scissors
   * Scissors beats Paper
   * Paper beats Stone
4. Scores are updated automatically.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/AwaisKhalil19/stone-paper-scissors.git
```

2. Open the project folder:

```bash
cd stone-paper-scissors
```

3. Open `index.html` in your browser.
